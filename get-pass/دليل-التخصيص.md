# 🎨 دليل التخصيص — غيّر كل شيء بنفسك

كل ما تحتاج تعديله موجود في ملفات قليلة. بعد أي تعديل: احفظ الملف، ثم **Ctrl + F5** في المتصفح.
> نصيحة: بعد تعديل ملفات `public/` لا تحتاج إعادة تشغيل الخادم. لكن إن عدّلت ملفات `src/` أعد التشغيل (`Ctrl+C` ثم `npm start`).

---

## 1) الألوان — ملف `public/styles.css` (أعلى الملف داخل `:root`)

غيّر القيمة بعد `:` (أكواد HEX). أهمها:

| المتغيّر | يتحكم في | القيمة الحالية |
|----------|----------|----------------|
| `--primary` | اللون الأساسي (الأزرار، الروابط، التحديد) | `#2563eb` |
| `--primary-dark` | لون الزر عند المرور عليه | `#1d4ed8` |
| `--primary-900` و `--primary-950` | تدرّج **القائمة الجانبية** (الغامقة) | `#1e3a8a` / `#172554` |
| `--bg` | لون الخلفية العام (خلف كل شيء) | `#f8fafc` |
| `--card` | لون الصناديق/البطاقات البيضاء | `#ffffff` |
| `--text` | لون النص الأساسي | `#111827` |
| `--muted` | لون النص الباهت (الوصف الصغير) | `#6b7280` |
| `--border` | لون حدود الصناديق | `#e5e7eb` |
| `--success` | الأخضر (نجاح/اعتماد/فعّال) | `#059669` |
| `--danger` | الأحمر (رفض/حذف/خطأ) | `#ef4444` |
| `--warning` | الكهرماني (تنبيه/طلب معلومات) | `#d97706` |

**ألوان شارات الحالة** (جديد/معتمد/مرفوض…): ابحث في نفس الملف عن `===== Badges =====` وعدّل `.s-new`, `.s-approved`, `.s-rejected`, `.s-active` … (كل واحدة فيها `background` و `color`).

---

## 2) الشعار والخلفية — مجلد `public/assets/`

- **الشعار:** استبدل ملف `logo.png` بشعارك (يظهر في الدخول والقائمة الجانبية).
- **الخلفية:** استبدل ملف `background.png` بخلفيتك.
> احتفظ بنفس الأسماء بالضبط. أفضل صيغة للشعار: PNG بخلفية شفافة.

---

## 3) الزوايا والظلال — `public/styles.css` داخل `:root`

| المتغيّر | يتحكم في |
|----------|----------|
| `--radius` | استدارة زوايا البطاقات (`0.75rem`) |
| `--radius-lg` | استدارة الأزرار والحقول (`0.5rem`) |
| `--shadow-sm/md/lg` | قوة الظلال |

---

## 4) الخط — السطر `font-family` في `public/styles.css`

ابحث عن `body {` ثم السطر `font-family:` وغيّر الخط. لتغيير خط جديد من Google Fonts، عدّل أيضاً رابط الخط في أعلى `public/index.html` (السطر الذي فيه `fonts.googleapis.com`).

---

## 5) الأسماء والعناوين

| ما تريد تغييره | المكان |
|----------------|--------|
| اسم النظام «PAMS» وسطر «نظام التصاريح والموافقات» | `public/index.html` ابحث عن `brand-name` و `brand small` |
| عنوان التبويب في المتصفح | `public/index.html` داخل وسم `<title>` |
| **عناصر القائمة الجانبية** (طلباتي، التصاريح، اللوحة…) | `public/app.js` ابحث عن `const NAV` |
| **أسماء الحالات** (جديد، قيد المراجعة، معتمد…) | `public/app.js` ابحث عن `const STATUS` — وللخادم: `src/services/workflow.js` ابحث عن `STATUS_LABELS` |
| **أسماء الأدوار** (مقدّم طلب، مراجِع، الدعم) | `public/app.js` ابحث عن `const ROLE` |
| نصوص نموذج الطلب وحقوله (تنبيه أبشر، الكفالة…) | `public/app.js` داخل دالة `viewNewRequest` |
| رؤوس أعمدة ملف Excel | التصاريح: `src/routes/permits.js` — الطلبات: `src/routes/requests.js` (ابحث عن `export.xlsx`) |

## 6) الصناديق (المستطيلات) ومواضعها — `public/styles.css`

| الصندوق | اسم الكلاس (Ctrl+F في styles.css) |
|---------|-----------------------------------|
| **القائمة الجانبية** (المستطيل الأزرق يمين) | `.sidebar` — لتغيير المسافات حوله عدّل `margin` و `border-radius` |
| **الشريط العلوي** (العنوان + التاريخ + الجرس) | `.topbar` |
| **منطقة المحتوى** (تنزيلها/رفعها) | `#content` — عدّل `margin-top` و `padding-bottom` |
| **اللافتة الصفراء** (تذكير التجديد) | `#banner` و `.banner.warn` |
| **البطاقات البيضاء** (كل صندوق محتوى) | `.card` |
| **بطاقات الأرقام** في اللوحة | `.kpi` |
| **الجداول** | `table`, `th`, `td` |
| **الأزرار** | `.btn` (و `.btn.success` / `.btn.danger` / `.btn.ghost`) |
| **الحقول والمدخلات** | `.field input, .field select, .field textarea` |
| **النافذة المنبثقة** (المودال) | `.modal`, `.modal-head` |
| **شارات الحالة** | `.badge` |
| **الإشعارات (الجرس)** | `.bell`, `.bell-badge` |

> مثال: لإنزال المحتوى أكثر، زِد الرقم في `#content { margin-top: 28px; }`.
> مثال: لإبعاد القائمة الجانبية عن الحواف، زِد `margin` في `.sidebar`.

## 6.5) المقاسات (الأحجام والعرض والمسافات) — `public/styles.css`

غيّر الرقم فقط. أهمها:

| تبي تغيّر | ابحث عن | الرقم |
|-----------|---------|-------|
| **عرض القائمة الجانبية (المستطيل اليمين)** | `.app { display: grid` | الرقم `264px` (الأول) |
| المسافة حول القائمة (تضغطها من فوق/تحت) | `.sidebar {` | `margin: 34px 16px 34px 0;` (+ `top` و `height`) |
| الحشو داخل القائمة | `.sidebar {` | `padding: 20px 16px;` |
| استدارة زوايا القائمة | `.sidebar {` | `border-radius: 22px;` |
| حجم الشعار داخل القائمة | `.brand-logo {` | `height: 46px;` / `max-width: 170px;` |
| خط عناصر القائمة | `.nav a {` | `font-size: 14px;` / `padding: 11px 14px;` |
| **عرض منطقة المحتوى** | `#banner, #content` | `max-width: 1180px;` |
| إنزال/رفع المحتوى | `#content {` | `margin-top: 28px;` / `padding-bottom: 36px;` |
| ارتفاع الشريط العلوي | `:root` → `--header-h:` | `64px` |
| حشو البطاقات | `.card {` | `padding: 22px;` |
| استدارة البطاقات | `:root` → `--radius:` | `0.75rem` |
| حجم الأزرار | `.btn {` | `padding: 10px 18px;` / `font-size: 15px;` |
| حجم الحقول | `.field input, .field select, .field textarea` | `padding: 10px 14px;` |
| بطاقات الأرقام | `.kpi {` و `.kpis {` | `padding: 20px;` / `minmax(160px, 1fr)` |
| عرض النافذة المنبثقة | `.modal {` | `max-width: 820px;` |
| حجم الجرس | `.bell {` | `width: 44px; height: 44px;` |

> الوحدات: `px` = بكسل ثابت، `rem` = نسبي للخط، `%` و `dvh` = نسبة من الشاشة.

## 7) إعدادات النظام (تحتاج إعادة تشغيل الخادم)

| الإعداد | المكان |
|---------|--------|
| نافذة التجديد (5 أيام) | `src/config.js` → `renewalWindowDays` (أو في ملف `.env`) |
| مدة التصريح الافتراضية | `src/config.js` → `defaultPermitDays` |
| المنفذ (4000) | `src/config.js` → `port` |
| نوع التحقق من الهوية (سعودي/عام) | `src/config.js` → `nationalIdMode` |
| **الحسابات الافتراضية** (البريد/كلمة المرور/الأسماء) | `src/db/seed.js` — بعد التعديل شغّل `npm run fresh` |

## 🌙 الوضع الداكن
- زر التبديل (🌙/☀️) في الشريط العلوي بجانب الجرس. يُحفظ اختيارك تلقائياً.
- ألوان الوضع الداكن في آخر `public/styles.css` تحت `body.dark { ... }` — غيّرها كما تشاء.

## ⚙️ أوامر مفيدة
```
npm start        # تشغيل النظام
npm run fresh    # تصفير قاعدة البيانات وإعادة الحسابات الافتراضية (أوقف الخادم أولاً)
```

## 🎯 القاعدة الذهبية
- ملفات **المظهر** كلها في: `public/styles.css` (الألوان/الصناديق) + `public/index.html` (الأسماء) + `public/assets/` (الصور).
- ملفات **المنطق** في: `src/` (القواعد والإعدادات والخادم).
