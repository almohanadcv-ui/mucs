## 1) تشغيل السيرفر محلياً على جهازك (للتجربة قبل الرفع)
في **CMD** على جهازك:
```cmd
cd "c:\Users\lenovo\Desktop\TSRE7"
npm start
```
- افتح المتصفح على: `http://localhost:4000`
- لإيقافه: اضغط **Ctrl + C** في نفس النافذة.
- ⚠️ شغّل نسخة واحدة فقط. لو قال «المنفذ مشغول»، أوقف القديمة بـ Ctrl+C أو:
  ```cmd
  taskkill /F /IM node.exe
  ```

## أوامر مفيدة على السيرفر
```bash
pm2 status              # حالة التطبيق (online/أخطاء)
pm2 logs pams           # عرض السجلّات الحيّة (Ctrl+C للخروج)
pm2 restart pams        # إعادة تشغيل
df -h /                 # المساحة المتبقية في القرص
du -sh /var/pams-data   # حجم بياناتك (قاعدة البيانات + الملفات)
```

## نسخة احتياطية كاملة (قاعدة البيانات + كل الملفات)
```bash
tar czf ~/pams-backup-$(date +%F).tar.gz /var/pams-data
```

## أين بياناتك على السيرفر
- قاعدة البيانات: `/var/pams-data/data/pams.db`
- الصور والملفات: `/var/pams-data/uploads/`
